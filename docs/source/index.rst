
jupyter_locuszoom
=====================================

Version: |release|

Jupyter Widget for LocusZoom


Quickstart
----------

To get started with jupyter_locuszoom, install with pip::

    pip install jupyter_locuszoom

or with conda::

    conda install jupyter_locuszoom
